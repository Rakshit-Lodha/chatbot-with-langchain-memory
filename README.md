Conversational Agent with Memory

An empathetic AI assistant that remembers conversation history using LangGraph's checkpoint system.

## Features
- **Persistent Memory**: Remembers user details across messages
- **Thread Management**: Supports multiple conversation threads
- **Empathetic Responses**: Configured personality for supportive replies
- **Conversation Control**: User can continue or exit gracefully

## Technical Highlights
- LangGraph InMemorySaver for state persistence
- Thread-based conversation tracking
- Agent framework with custom system prompts
- Clean conversation loop architecture

## Tech Stack
- LangChain + LangGraph
- InMemorySaver (checkpoint system)
- OpenAI GPT-3.5-turbo
- Python
